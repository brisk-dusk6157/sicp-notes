1. Section 1.2.6. Example: Testing for Primality

1.1. Searching for divisors

The \(O (\sqrt {n})\) version finds the smallest divisor of n, by trying 2,3,…,\(\left\lfloor \sqrt {n}\right\rfloor\). If none of them is a divisor, then n must be the only divisor of itself, and so is prime. 

#lang sicp

(define (smallest-divisor n)
  (find-divisor n 2))

(define (square x) (* x x))

(define (divides? d n) (= 0 (remainder n d)))

(define (find-divisor n test-divisor)
  (cond ((> (square test-divisor) n) n)
	((divides? test-divisor n) test-divisor)
	(else (find-divisor n (inc test-divisor)))))
(define (prime? n)
  (= (smallest-divisor n) n))
(prime? 169)
(prime? 167)

1.2. The Fermat test

Format's Little Theorem:

If \(n\) is a prime and \(a\) is any positive integer with \(a \lt n\), then \(a^n \equiv a \pmod n\).

One consequence is that if \(n\) is not a prime, then most of the numbers \(a \lt n\) will not satisfy the property.

Suggested algorithm for testing primality (the Fermat test): Given a number \(n\), pick a random number \(a \lt n\), and if \(a^n \equiv a \pmod n\), then n is probably a prime. Pick another \(a\) and try again. By trying more and more values for a, we can increase our confidence in the result. 

#lang sicp

(define (square x) (* x x))

(define (expmod base exp m)
  (cond ((= exp 0) 1)
	((even? exp)
	 (remainder (square (expmod base (/ exp 2) m))
		    m))
	(else
	 (remainder (* base (expmod base (- exp 1) m))
		    m))))

(define (fermat-test n)
  (define (try-it a)
    (= (expmod a n n) a))
  (try-it (inc (random (dec n)))))

(define (fast-prime? n times)
  (cond ((= times 0) #t)
	((fermat-test n) (fast-prime? n (dec times)))
	(else #f)))

(fast-prime? 561 10)

(runtime) is a primitive in Scheme that returns the amount of time the system? has been running. 

#lang sicp

(define (smallest-divisor n)
  (find-divisor n 2))

(define (square x) (* x x))

(define (divides? d n) (= 0 (remainder n d)))

  (cond ((> (square test-divisor) n) n)
	((divides? test-divisor n) test-divisor)
	(else (find-divisor n (inc test-divisor)))))

(define (prime? n)
  (= (smallest-divisor n) n))

(define (timed-prime-test n)
  (start-prime-test n (runtime)))

(define (start-prime-test n start-time)
  (if (prime? n)
      (report-prime n (- (runtime) start-time)))
  (prime? n))

(define (report-prime n elapsed-time)
  (newline)
  (display n)

	     (search-for-primes (+ n 2)

1.4. Exercise 1.23

find-divisor does a lot of extra work, we cut it at least in half by skipping even numbers 

#lang sicp

(define (smallest-divisor n)
  (find-divisor n 2))

(define (square x) (* x x))

(define (divides? d n) (= 0 (remainder n d)))

(define (next d) (if (= d 2) 3 (+ d 2)))

(define (find-divisor n test-divisor)
  (cond ((> (square test-divisor) n) n)
	((divides? test-divisor n) test-divisor)
	(else (find-divisor n (next test-divisor)))))
